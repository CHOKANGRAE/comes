    기능 테스트

    기능테스트는 Product SDLC 전반에
    폭넓게 적용되는 테스트 방법으로 요구사항에
    부합하고 버그가 없는 Product를 완성하는
    중요한 테스트 입니다.

    성능 테스트

    컴즈의 성능테스트는 적절한 리소스로
    시스템을 구성하고 보증하여
    빠른 서비스를 제공할 수 있도록
    테스트 지원합니다.

    API 테스트

    컴즈는 API 테스트의 차별화된 전략으로
    단순 호출과 응답에 대한 테스팅을 넘어
    시나리오 형태로 테스트 Suit을 작성하여
    기능테스트의 리소스를 절감할 수 있도록
    지원합니다.
